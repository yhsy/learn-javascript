任务4：函数 pow(x,n)

写一个函数 pow(x,n)，返回 x 的 n 次方。换句话说，将 x 与自身相乘 n 次，返回最终结果。

        pow(3, 2) = 3 * 3 = 9
        pow(3, 3) = 3 * 3 * 3 = 27
        pow(1, 100) = 1 * 1 * ...*1 = 1

P.S. 在这个任务中，函数应该只支持自然数 n：从 1 开始的整数。

代码：

        const num1 = prompt('请输入x的值', '')
        const num2 = prompt('请输入n的值', '')

        if(num2< 1) {
          alert('n的值输入错误，n的类型必须是正整数')
        } else {
          alert(pow(num1, num2));
        }
  
        
  
        function pow(x, n) {
          // console.log(x, n)
          let sum = 1;
          for (let i = 0; i < n; i++) {
            sum *= x
          }
          return sum
        }
