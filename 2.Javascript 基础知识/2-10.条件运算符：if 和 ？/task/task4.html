任务4：使用 '?' 重写 'if' 语句

使用条件运算符 '?' 重写下面的 if 语句：

            let result;
            
            if (a + b < 4) { result='Below' ; } else { result='Over' ; }

三元运算符方式：

                // 默认值: 假设a=1,b=2
                const a = 1;
                const b = 2;
                let result = ( a + b < 4) ? 'Below' : 'Over';
                
                alert(result)
