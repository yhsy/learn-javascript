任务5：使用 '?' 重写 'if..else' 语句

使用多个三元运算符 '?' 重写下面的 if..else 语句。

            let message;
            
            if (login == 'Employee') {
                message = 'Hello';
            } else if (login == 'Director') {
                message = 'Greetings';
            } else if (login == '') {
                message = 'No login';
            } else {
                message = '';
            }

三元运算符方式：用户名(Employee,Director)

                let login = prompt('请输入用户名','');
                let message = (login == 'Employee') ? 'Hello' : (login == 'Director') ? 'Greetings' : (login == '') ? 'No login' : '';
                alert (message)
