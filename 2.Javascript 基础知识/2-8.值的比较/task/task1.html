以下表达式的执行结果是？

        5 > 4                       => true     原因：数字间比较大小，显然得 true。
        "apple" > "pineapple"       => false    原因：按词典顺序比较，得 false。"a" 比 "p" 小。
        "2" > "12"                  => true     原因：与第 2 题同理，首位字符 "2" 大于 "1"。
        undefined == null           => true     原因：null 只与 undefined 互等。
        undefined === null          => false    原因：严格相等模式下，类型不同得 false。
        null == "\n0\n"             => false    原因：与第 4 题同理，null 只与 undefined 相等。
        null === +"\n0\n"           => false    原因：不同类型严格不相等。
