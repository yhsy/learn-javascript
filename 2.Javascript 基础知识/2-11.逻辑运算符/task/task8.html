任务8：一个关于"if"的问题

下面哪一个 alert 将会被执行？

if(...) 语句内表达式的结果是什么？

        if (-1 || 0) alert( 'first' );
        if (-1 && 0) alert( 'second' );
        if (null || -1 && 1) alert( 'third' );
    
        if (-1 || 0) alert( 'first' );  => 打印 first
        if (-1 && 0) alert( 'second' );  => 不打印
        if (null || -1 && 1) alert( 'third' ); => 打印 third
