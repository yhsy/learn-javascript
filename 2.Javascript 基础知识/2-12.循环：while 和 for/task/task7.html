任务6：输出素数（prime）

  • 素数定义
  • 1.大于 1 且不能被除了 1 和它本身以外的任何数整除的整数叫做素数。
  • 2.换句话说，n > 1 且不能被 1 和 n 以外的任何数整除的整数，被称为素数。
  • 3.例如，5 是素数，因为它不能被 2、3 和 4 整除，会产生余数。

写一个可以输出 2 到 n 之间的所有素数的代码。

当 n = 10，结果输出 2、3、5、7。

P.S. 代码应适用于任何 n，而不是对任何固定值进行硬性调整。

代码：

            // let n = prompt('请输入大于2的整数', '');
            let n = 10;

            nextContinue:
            for (let i = 2; i <= n; i++) { 
                for (let j=2; j < n; j++) { 
                    if (i % j===0) { continue nextContinue } 
                } 
                alert(i) // 输出素数 
            }
