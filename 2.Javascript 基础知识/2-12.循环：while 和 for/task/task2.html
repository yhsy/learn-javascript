任务2：while 循环显示哪些值？

对于每次循环，写出你认为会显示的值，然后与答案进行比较。

以下两个循环的 alert 值是否相同？

  • 1.前缀形式 ++i

                        let i = 0;
                    while (++i < 5) alert( i );

  • 2.后缀形式 i++

                        let i = 0;
                    while (i++ < 5) alert( i );
                
            总结果: 不相同
            解题: 
            1.前缀形式：结果是 1,2,3,4
            2.后缀形式：结果是 1,2,3,4,5
