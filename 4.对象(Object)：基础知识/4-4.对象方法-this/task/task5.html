任务5：链式调用

有一个可以上下移动的 ladder 对象：

      let ladder = {
        step: 0,
        up() {
          this.step++;
        },
        down() {
          this.step--;
        },
        showStep: function() { // 显示当前的 step
          alert( this.step );
        }
      };

现在，如果我们要按顺序执行几次调用，可以这样做：

      ladder.up();
      ladder.up();
      ladder.down();
      ladder.showStep(); // 1

修改 up，down 和 showStep 的代码，让调用可以链接，就像这样：

      ladder.up().up().down().showStep(); // 1

代码：

      let ladder = {
        step: 0,
        up() {
          this.step++;
          return this;
        },
        down() {
          this.step--;
          return this;
        },
        showStep: function() { // 显示当前的 step
          alert( this.step );
          return this;
        }
      };
