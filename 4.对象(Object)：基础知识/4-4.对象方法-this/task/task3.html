任务3：在对象字面量中使用 "this"

这里 makeUser 函数返回了一个对象。

访问 ref 的结果是什么？为什么？

      function makeUser() {
        return {
          name: "John",
          ref: this
        };
      };
      
      let user = makeUser();
      
      alert( user.ref.name ); // 结果是什么？

代码解析：

      结果：undefined;
      原因：因为设置 this 的规则不考虑对象定义，只有调用那一刻才重要。
