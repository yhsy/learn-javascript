任务2：检查空对象

写一个 isEmpty(obj) 函数，当对象没有属性的时候返回 true，否则返回 false。

应该像下面这样：

      let schedule = {};

      alert(isEmpty(schedule) ); // true
      
      schedule["8:30"] = "get up";
      
      alert( isEmpty(schedule) ); // false

代码：

        let schedule = {};

        alert(isEmpty(schedule)); // true
  
        schedule["8:30"] = "get up";
  
        alert(isEmpty(schedule)); // false
  
        function isEmpty(obj) {
          // 方法1(JSON.stringify)
          // if (JSON.stringify(obj) === '{}') {
          //   return true;
          // }
          // return false;
  
          // 方法2(ES6方法)
          const len = Object.keys(obj).length;
          if (len === 0) {
            return true;
          }
          return false;
        }
